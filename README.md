## 説明

趣味の合う友達探しのサービス

## 使い方

## カテゴリー選択
１. 新規会員登録後にマイページ編集へ

2. マイページ編集ページでカテゴリーをチェックする。


3. TOPへ遷移し、カテゴリーを選択

## ダイレクトメッセージの流れ

１. お互いがフォローし合わないとダイレクトメッセージが送れないません。<br>
   まずはユーザーをフォローしましょう。

2. フォローされたら通知されます。


3. 相互フォローになったら、相手のプロフィールに『DMを送ろう』ボタンが表示されます。


4. そして、ダイレクトメッセージの完了です。
